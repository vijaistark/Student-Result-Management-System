Smart Student Result Management System - Directory Structure
================================================================

Student-Result-Management-System-1/
│
├── admin/                          # Admin Module
│   ├── dashboard.php              # Admin main dashboard (statistics, queries)
│   ├── view_query.php             # View and respond to student queries
│   └── view_results.php           # View all results (read-only)
│
├── staff/                          # Staff Module
│   ├── dashboard.php              # Staff main dashboard (post marks, statistics)
│   ├── post_marks.php             # Post marks for students (once-only)
│   └── view_marks.php             # View posted marks (read-only)
│
├── student/                        # Student Module
│   ├── dashboard.php              # Student dashboard (view results, queries)
│   └── submit_query.php           # Submit query handler
│
├── assets/                         # Static Assets
│   └── css/
│       └── style.css              # Main stylesheet (modern, responsive)
│
├── config/                         # Configuration Files
│   ├── config.php                 # Main config (session, constants, includes)
│   └── database.php               # Database connection handler
│
├── database/                       # Database Files
│   └── schema.sql                 # Complete database schema with sample data
│
├── includes/                       # Common Includes
│   ├── functions.php              # Helper functions (sanitize, calculate, etc.)
│   ├── header.php                 # Common header component
│   └── footer.php                 # Common footer component
│
├── utils/                          # Utility Scripts
│   └── generate_password_hash.php # Password hash generator tool
│
├── index.php                       # Login page (entry point)
├── logout.php                      # Logout handler
├── .htaccess                       # Apache configuration
│
├── README.md                       # Complete documentation
├── INSTALLATION.md                 # Installation guide
├── PROJECT_SUMMARY.md              # Project overview and features
└── DIRECTORY_STRUCTURE.txt         # This file

================================================================
Key Entry Points:
================================================================

1. Login:        index.php
2. Admin:        admin/dashboard.php
3. Staff:        staff/dashboard.php
4. Student:      student/dashboard.php
5. Logout:       logout.php

================================================================
Database Schema: database/schema.sql
================================================================

Tables:
- users       (admin, staff, student accounts)
- students    (student information)
- subjects    (available subjects)
- marks       (student marks - UNIQUE per student+subject)
- queries     (student queries with admin responses)
